# StyleFit 3D 产品需求设计

## 1. 产品定位

StyleFit 3D 是一个面向 iOS 和 Android 的智能穿搭应用。用户上传本人照片并填写身高、体重、版型偏好后，系统生成个人 3D 试穿模特；用户也可以上传衣服照片建立 3D 衣橱，应用围绕衣橱管理、3D 试穿、穿搭编辑和商品推荐生成搭配方案。天气和气温只是推荐排序的上下文因素，不是主要功能入口。

## 2. 目标用户

- 想降低每日穿搭决策成本的通勤用户。
- 经常网购但担心尺码、比例和上身效果的人。
- 希望管理自己衣橱并减少重复购买的人。
- 内容创作者、买手、造型师等需要快速生成搭配方案的人。

## 3. 核心用户流程

1. 新用户填写身高、体重、预算、版型偏好和喜欢的颜色。
2. 用户上传本人照片，系统生成带身体比例的试穿模特。
3. 用户上传衣服照片，选择品类，系统生成衣服 3D 预览并加入衣橱。
4. 用户选择穿着场景，系统根据衣橱、个人数据和环境上下文生成今日穿搭模型。
5. 用户在试穿页直接编辑上装、下装、外套、鞋履和配饰槽位，3D 预览即时更新。
6. 用户查看推荐理由、搭配适配分、保暖权重和单品层级。
7. 系统根据衣橱缺口、身体数据、预算、场景和环境上下文推荐网上商品。

## 4. MVP 范围

当前版本已经开始开发以下能力：

- Expo + React Native + TypeScript 跨端工程，支持 iOS/Android。
- 本人照片上传入口，用于个人试穿模特。
- 身高、体重、版型偏好输入。
- 3D 风格试穿预览，展示上装、下装、外套、鞋履、配饰的层级效果。
- 当前穿搭槽位编辑器，支持在试穿页直接切换同类衣服。
- 衣服照片上传入口，支持选择上装、下装、外套、鞋履、配饰。
- 基于衣橱、场景、个人数据和天气上下文的本地推荐算法。
- 基于预算、场景、衣橱缺口和颜色偏好的商品推荐列表。

## 5. 后续功能

- 真实人体建模：从多角度照片或短视频生成可调身材模型。
- 真实服装建模：服装分割、材质识别、褶皱模拟、尺码估计。
- 环境上下文服务：接入定位、城市搜索、体感温度、降雨概率等，用于推荐权重而不是作为独立核心功能。
- 电商服务：接入商品搜索、库存、尺码表、佣金链接、价格追踪。
- AI 造型师：解释推荐逻辑，生成多套备选风格。
- 社交分享：导出穿搭图、保存日历、收藏搭配。
- 隐私保护：照片加密上传、可删除模型、敏感数据最小化存储。

## 6. 关键指标

- 首次生成试穿模型完成率。
- 每周衣橱上传件数。
- 每日穿搭推荐点击率。
- 商品推荐点击率和收藏率。
- 试穿模型生成耗时和失败率。
- 用户删除照片/模型流程成功率。

## 7. 风险和约束

- 单张照片无法稳定恢复真实 3D 身体和服装厚度，生产环境应支持多角度采集。
- 用户照片、身体数据属于敏感信息，需要明确授权、加密存储和删除机制。
- 电商推荐需要处理商品版权、价格变化、库存、尺码表准确性和广告标识。
- 真实物理布料模拟成本较高，MVP 可先使用估计模型，逐步升级到服务端生成。
